# Виконання SQL-операцій
На цій лабораторній работі ми навчилися виконанню SQL-запитів для керування даними фільмотеки. Запити охоплюють основні операції маніпулювання даними: вибірку, додавання, оновлення та видалення.


## Операції SELECT (Вибірка даних)

- SELECT 1: Мета цього запиту — отримати список фільмів, відсортованих за роком випуску. Очікуваний результат: хронологічний список назв та років від найстаріших до новітніх. Запит виконано успішно.

- SELECT 2: Мета — знайти бюджетні плани підписок вартістю до 9.99. Очікуваний результат: перелік підписок зі зниженою ціною. Запит виконано успішно.

- SELECT 3: Мета — переглянути всі успішні платежі від нових до старих. Очікуваний результат: хронологічна стрічка успішних транзакцій. Запит виконано успішно.

- SELECT 4: Мета — знайти всі фільми за участю актора Тома Холланда. Очікуваний результат: список фільмів, де грав цей актор. Запит виконано успішно.

- SELECT 5: Мета — знайти всі фільми режисерки Грети Гервіг. Очікуваний результат: фільмографія вказаного режисера. Запит виконано успішно.

- SELECT 6: Мета — відфільтрувати фільми для дорослої аудиторії. Очікуваний результат: список фільмів з віковим обмеженням 16+, 18+ або 21+. Запит виконано успішно.

- SELECT 7: Мета — отримати ТОП-10 останніх новинок сервісу. Очікуваний результат: перелік з 10 фільмів з найпізнішим роком випуску. Запит виконано успішно.

## Операції INSERT (Додавання даних)

- INSERT 1: Мета — додати інформацію про нового режисера. Очікуваний результат: Стівен Спілберг з'являється у таблиці director з новим ID. Запит виконано успішно.

- INSERT 2: Мета — додати новий фільм у каталог. Очікуваний результат: фільм "Interstellar" з'являється у таблиці film. Запит виконано успішно.

- INSERT 3: Мета — зареєструвати нового клієнта в системі. Очікуваний результат: новий користувач Емілі Джонсон з'являється у таблиці customer. Запит виконано успішно.

## Операції UPDATE (Оновлення даних)

- UPDATE 1: Мета — змінити віковий рейтинг фільму "Dune". Очікуваний результат: поле age_restriction для "Dune" оновлюється на '16+'. Запит виконано успішно.

- UPDATE 2: Мета — оновити контактну пошту клієнта. Очікуваний результат: змінена електронна адреса (email) для вказаного користувача. Запит виконано успішно.

- UPDATE 3: Мета — достроково припинити (скоротити термін) підписку користувача. Очікуваний результат: встановлення нової дати end_date для підписки клієнта. Запит виконано успішно.

## Операції DELETE (Видалення даних)

- DELETE 1: Мета — видалити актора з конкретного фільму. Очікуваний результат: розірвання зв'язку у проміжній таблиці film_actor. Запит виконано успішно.

DELETE 2: Мета — видалити певний жанр у фільму. Очікуваний результат: розірвання зв'язку у проміжній таблиці film_genre. Запит виконано успішно.

DELETE 3: Мета — очистити старі невдалі платежі. Очікуваний результат: видалення з таблиці payment усіх записів зі статусом FALSE, зроблених до 2025 року. Запит виконано успішно.
